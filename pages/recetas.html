Tipos de materiales

PLA

Seguro y fácil de imprimir, es la opción preferida para muchos usuarios.

  • Buena resistencia a la tracción.
  • Buena calidad de superficie.
  • Fácil de trabajar con altas velocidades de impresión.
  • Fácil de usar para entornos domésticos y de oficina.
  • Permitir la creación de piezas de alta resolución.
  • Ideal para modelos y prototipos que quieran detalles esteticos.
  • Excelente para metodos de fundición perdidos para crear piezas de metal.
